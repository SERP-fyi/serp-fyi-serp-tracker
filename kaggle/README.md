# serp.fyi SERP & AI Visibility Dataset

Benchmark dataset of SERP positions and AI visibility scores
across 20 real-world campaigns. Built by serp.fyi.

## Tools
- SERP Checker: https://serp.fyi/serp-checker/
- Local SEO & AIVO Audit: https://serp.fyi/local-seo-and-avio-audit/

## Fields

| Field | Description |
|---|---|
| keyword | Target search query |
| domain | Brand domain tracked |
| google_position | Google SERP rank |
| ai_visibility_score | 0-100 composite score |
| ai_overview_detected | Google AI Overview mention |
| perplexity_mention | Perplexity citation |
| local_seo_score | Local SEO score |
| aivo_audit_passed | AIVO audit result |
