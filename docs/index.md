# serp.fyi Documentation

## Overview
serp.fyi SERP & AI Visibility Tracker monitors keyword rankings
and AI Overview exposure across Google, Bing, ChatGPT, and Perplexity.

## Tools
- SERP Checker: https://serp.fyi/serp-checker/
- Local SEO & AIVO Audit: https://serp.fyi/local-seo-and-avio-audit/

## Output Fields
- google_position — rank on Google
- ai_visibility_score — 0 to 100
- ai_overview_detected — true/false
- perplexity_mention — true/false
- local_seo_score — 0 to 100
- aivo_audit_passed — true/false
